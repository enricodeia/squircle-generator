What Makes Squircles Special?

Standard border radius creates a perfect quarter-circle at each corner, but this can look mechanically rigid in UI design. Squircles use a more sophisticated curve that varies along its length, creating a more organic, visually pleasing shape.

The curve of a squircle doesn't maintain a constant radius. Instead, it uses a continuous function that produces a curve with varying curvature - tighter near the corner point, gentler as it transitions to the straight edges.

Where Squircles Are Used

Apple pioneered the use of squircles in UI design, using them for iOS app icons, interface elements, and hardware designs. They create a softer, more organic feel than standard rounded rectangles.

Today, many design systems incorporate squircle-like geometry for components like buttons, cards, and panels to create a more harmonious, natural appearance.

Understanding Squircles in UI Design

        
        

          

            The squircle is a mathematical shape that blends the properties of a square and a circle to create 
            a visually balanced, organic form. In modern UI design, squircles have become increasingly important 
            for creating interfaces that feel natural and refined.
          

          
          

            

              
The Mathematical Difference

              

                A standard rounded rectangle uses a perfect quarter-circle at each corner with a constant radius. 
                A squircle uses a superellipse-based curve where the radius of curvature varies continuously along the path.
              

              

                The formal equation for a superellipse is |x/a|ⁿ + |y/b|ⁿ = 1, where n controls the "squircleness." 
                iOS specifically uses a modified curve with Bézier control points to create their unique squircle shape.
              

            

            
            

              
Why Squircles Matter

              

                Human perception is sensitive to subtle curve differences. The organic, varying curvature of squircles 
                feels more natural and visually pleasing than the mathematically perfect but rigid circular arcs.
              

              

                This subtle difference contributes significantly to the perceived quality and refinement of an interface. 
                It's one of many small details that together create a premium, polished user experience.
              

            

          

        

        
        

          
Applications in Modern Design

          

            

              

              
App Icons

              

                iOS app icons use the squircle shape to create a consistent, cohesive visual language. 
                This subtle curve creates a more organic feel than standard rounded squares.
              

            

            
            

              
Button

              
Buttons & Controls

              

                Buttons with squircle corners feel more inviting and natural. The varied curve creates a subtle 
                improvement in perceived quality and design sophistication.
              

            

            
            

              

                

                

              

              
Cards & Containers

              

                Content cards and containers benefit from squircle corners by creating a softer, 
                more harmonious frame for content display.
              

            

          

        

        
        

          
Implementation Approaches

          

            There are several ways to implement squircles in modern web and app interfaces:
          

          
    
            
  • 
              SVG Path: The most precise method, using Bézier curves to create the exact squircle shape.
            
    • 
            
  • 
              CSS clip-path: Using the path() function to clip a rectangular element into a squircle shape.
            
    • 
            
  • 
              Border radius approximation: Some design systems use different border-radius values for each corner to approximate the squircle effect.
            
    • 
            
  • 
              Custom masking: Creating a squircle mask and applying it to UI elements.
